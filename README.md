# modern-ie-automate

modern.ie auto launch script.

## support os
- OS X

## requirements

- node (>= v6.0.0 )
- Virtualbox
    - https://www.virtualbox.org/
- Vagrant
    - https://www.vagrantup.com/

## Setup

```
$ git clone
$ npm install
```

## Usage
```
$ npm start
```
only!

## Note
The size of one box will be 3-4 Gbytes.
Be careful of the free space on the disk.
